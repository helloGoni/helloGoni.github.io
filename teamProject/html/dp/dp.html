동적 계획 알고리즘이란

동적 계획 알고리즘(Dynamic Programming Algorithm)이란 최적화 문제를 해결하는 알고리즘 중에 하나이다. 입력 크기가 작은 부분문제들을 해결하고, 그 부분문제들의 해를 이용하여 최종적으로 원래 주어진 입력의 문제를 해결한다.

분할 정복 알고리즘과 어느 정도 유사하면서도 다르다. 분할정복 알고리즘은 A가 B와 C로 분해된다면 A라는 문제를 푸는데는 무조건 B와 C의 해를 요구한다. 그러나 동작 계획 알고리즘의 경우 이전에 구한 다른 부분 문제를 푸는데 사용될 수 있고, 이것에 대한 순서는 대부분의 경우 뚜렷하게 보이지 않는다. 이를 함축적 순서(implicit order)라고 한다.

출처

양성봉, 『알기 쉬운 알고리즘』, 생능출판사, p162-163.
