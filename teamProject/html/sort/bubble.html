버블 정렬BubbleSort



버블 정렬(Bubble Sort)은 이웃하는 숫자를 비교하여 작은 수를 앞쪽으로 이동시키는 과정을 반복하여 정렬하는 알고리즘이다.

의사 코드

입력: 크기가 n인 배열
출력: 정렬된 배열 A
  1. for pass = 1 to n-1
  2.  for i = 1 to n-pass
  3.   if A[i-1] > A[i]
  4.    swap(A[i-1], A[i])
  5. return 배열A

시간 복잡도

pass = 1에서 n-1번 비교하고, pass = 2에서 n-2번 비교하고... pass = n-1에서 1번 비교한다. 그러므로 총 비교 횟수는 n(n-1)/2이다. if 조건이 True일때 자리 바꿈은 O(1)시간이 걸린다. 그러므로 버블 정렬 알고리즘의 시간 복잡도는 O(n(n-1)/2)×O(1) -> O(n2)이다.
